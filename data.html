DATA SF

The data set was taken from Data SF and shows the Film Locations in San Francisco .

It collects data through a listing of filming locations of movies shot in San Francisco starting from 1924 to September 2019 (Access dataset on April 2020)

All data are under Public Domain Dedication and License (PDDL).

From Open Data Commons webpage on PDDL:

You are free to:

1.To Share: To copy, distribute and use the database.

2.To Create: To produce works from the database.

3.To Adapt: To modify, transform and build upon the database. --Data SF license information

The data itself is divided into 3 columns and many rows. (scrollable textbox area)

Field Name Data Type Definition
Title Text Movie title.
Release Year Number Date movie was released.
Locations Text Location of filmning.
Fun Facts Text Fun facts about the filming.
Production Company Text Company that was producing the film.
Distributer Text Distributer of the movie.
Director Text Director of the film.
Writer Text Writer of the movie.
Actor 1 Text First actor in the movie.
Actor 2 Text Second actor in the movie.
Actor 3 Text Third actor in the movie.

TMDB API

The Movie Database (TMDB) offers a powerful API service that is free to use. Utilized the GET Movie Details query to recieve a JSON object with information about a movie.

JSON tags used for this project:

  • budget
  • genres
  • imdb_id
  • realease_date
  • revenue
  • title
  • vote_average

PROCESSING/FILTERING/WRANGLING

The "Film_Locations_in_San_Francisco.csv" Dataset was loaded into a python script to be processed. The script eleminated any repeated titles along with movies before the year 2000. Once the filtered data has been collected, the script utilizes the TMDB API to request for movie details as a JSON object. Movies that were not found in API were disgarded from the collected movies. Once done, the script prints out each movie along with the targeted information from the JSON objects. Using the terminal to run the script the output was piped out into a csv file "sunburst.csv"
